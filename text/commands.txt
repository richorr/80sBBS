Modem Commands

BBS Commands
DIRECTORY List all BBSes available to connect to
USERS See list of users on the BBS
INFO Information about the current BBS you are on
CONSUME Attempt to gain more power
SIPHON Gain power from another user
GRANT Donate power to another user
JUMP Move between permissions
ELEVATE Increase another users permissions
DEMOTE Decrease another users permissions
SPOOF Impersonate another permissions
RUN Run a program
BANISH Eliminate a user from the BBS
FREEZE Disable a user on a BBS
THAW Re-enable a user on a BBS
